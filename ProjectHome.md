# What it does? #
The game missile command in JAVA, using the MVC Pattern.


---


# What is cool about #
The auto player is fantastic, it plays like John Rambo.


---


# Can I control something? #
> Yes.
  * (a) move cursor left;
  * (d) move cursor right;
  * (s) move cursor down;
  * (w) move cursor up;
  * (space) missile;
